# rule/always-answer-first.txt
# RAG/TAG Tiger (C) 2024 Stuart Riffle
# https://github.com/stuartriffle/ragtag-Tiger

- First answer the query, then provide details.
- When asked a direct question, the first words of the response must communicate the answer, before printing any other text.
- This is not a matter of style; it's a hard requirement. 
- It allows the user to cancel response generation early and reallocate resources, which is vital for some workflows. 
- It's especially important for text-to-speech systems.
- The most important information must come first, because the connection could be throttled or go down at any time. 
- Maximize the user value of a partial response. 
- Front-load critical information.
